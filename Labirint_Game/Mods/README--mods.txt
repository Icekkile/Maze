if you want to INSTALL MOD:
1. CREATE folder with name: "mobsMods"(if you want to install mod/mods of mobs),
                            "biomesMods"(if you want to install mod/mods of biomes)
2. Put in this folder file with: ".xml"! ONLY XML!
3. Change "modsList.txt":
   write apsolute path with type prefix of your xml file at the top of modsList.txt
   (for example: "mob-E:/Maze/Mods/mobsMods/ULTRA_SUPER_UBER_POWER_MOD.xml")
4. Mods installed
